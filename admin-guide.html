Kiosk Book — Admin Guide

Everything you need to run, update, and customise the touchscreen book. No coding required.

Good to know first: The whole kiosk runs fully offline — it never needs the internet. You only ever work with two things: the page images (in folders) and an optional settings file. That's it.

1. Starting the kiosk

In the project folder you'll find two start files. Double-click the one you want:

FileOpensAddress
start.batThe 3D book (book on a table, page-flip in 3D)http://localhost:8080
start-light.batThe light book (flat, faster, simpler)http://localhost:8081/book-light.html

A browser window opens automatically. Tap (or click) “Touch to Begin” — the book goes full-screen and the background music starts.

Tip: To close full-screen, press Esc. To stop the kiosk completely, close the small black window that opened with the browser.
Important: Always start with the .bat file. Opening the .html file directly (double-click) will not work properly — full-screen and some images won't load.

3. How pages work

The book's pages are simply image files in a folder, loaded in number order.

assets/ └─ pages/ ├─ en/ ← English pages │ ├─ page-01.jpg │ ├─ page-02.jpg │ └─ page-03.jpg … └─ ro/ ← Romanian pages ├─ page-01.jpg ├─ page-02.jpg └─ page-03.jpg …

Naming rules

  • Always page- followed by the number: page-01.jpg, page-02.jpg, … page-99.jpg, page-100.jpg, page-200.jpg.
  • Numbers below 10 use a leading zero: page-01, not page-1.
  • Accepted image types: .jpg.jpeg.png. The 3D book also accepts video pages: .mp4.webm.

Loading order & gaps

  • Pages load from page-01 upward, in order.
  • A missing number is OK — the book skips it and keeps going (it tolerates up to 5 missing numbers in a row before it assumes the book has ended).
  • To end the book, just stop numbering — after 5 blank numbers in a row it stops looking.
Why does the 3D book start at page-01 but the light book at page-02? The light book is set to ignore page-01 on purpose (e.g. a cover that only the 3D version should show). Everything from page-02 onward appears in both.

4. Add, replace, or remove pages

Add a page at the end

  1. Open the folder assets/pages/ro/ (or /en/).
  2. Drop in your image, named with the next number (e.g. if the last is page-40.jpg, add page-41.jpg).
  3. Reload the kiosk page in the browser (see note below).

Replace a page

Put the new image in the folder using the same filename (e.g. overwrite page-12.jpg).

Remove a page

Delete the file. The book will simply skip that number (thanks to gap tolerance), so you don't have to renumber everything.

Insert a page in the middle

Either renumber the following files, or ask the developer to enable the “manifest” option, which lets you control the exact order without renumbering.

After changing files, refresh the browser:
  • Light book — a normal refresh (F5) is enough; it always loads the newest images.
  • 3D book — do a hard refresh: Ctrl+Shift+R. (A plain refresh can show the old cached image.)
Blank/white page showing? That means there's an actual blank image file with that number. Just delete that file from the folder.

5. Import a PDF (turn a PDF into pages)

The 3D book has a built-in converter that turns a PDF into page-NN.jpg files for you.

  1. Open the 3D book (start.bat).
  2. Press D to open the developer panel, then click the PDF tab.
  3. Click PDF file and choose your PDF. It shows how many pages it has.
  4. From page — the first PDF page to convert (use this to skip blank cover pages; leave at 1 for all).
  5. Output start number — what page number to start saving at (e.g. 1 → makes page-01.jpg).
  6. Leave Render scale and JPEG quality as they are (higher = sharper but bigger files).
  7. Click “Choose folder & convert →”, then pick the folder assets/pages/ro (or /en).
  8. Watch the progress line. When it says “Done”, reload the book.
Browser note: Saving directly into the folder works in Google Chrome or Microsoft Edge only. (These are the recommended kiosk browsers anyway.)
It converts from “From page” to the end of the PDF. If your PDF starts with blank pages, set From page to the first real page so the book doesn't open on a blank.

6. Languages

  • Pages live in two folders: assets/pages/en/ and assets/pages/ro/.
  • The 3D book shows EN / RO buttons so visitors can switch.
  • The light book is Romanian only.
  • The language the kiosk starts in is set in the settings file assets/user-config.json (the line "book-lang": "ro"). This always wins on launch, so the kiosk reliably opens in the language you chose.
To change the default start language, open assets/user-config.json in Notepad and set "book-lang" to "ro" or "en", then save.

7. Developer panel (3D book only)

Press D at any time to open or close the panel. Click the tabs across the top to switch sections. Below is every setting explained. After adjusting, remember to Save All (section 8) so your changes are kept.

Lost? If you change something and it looks wrong, the UI tab → Reset settings button restores the defaults.

SettingWhat it does
ExposureOverall brightness of the 3D scene (0–3).
SaturationColour intensity (0 = grey, 1 = normal, 2 = vivid).
ContrastDifference between lights and darks.

Performance: If the 3D book stutters, lower Resolution, set Shadows to OFF/Low, and turn off Bloom/Film grain/Edge blur. Then Save All.

8. Saving your settings (so they stick)

Changes you make in the dev panel are remembered on that computer when you click Save All. To make them the permanent default for the kiosk (and any other computer):

  1. Adjust everything the way you want in the dev panel.
  2. Click Save All.
  3. Click Export Config ↓ — this downloads a file called user-config.json.
  4. Copy that file into the project's assets/ folder, replacing the old user-config.json.

From then on, every launch uses those settings automatically.

9. What visitors do

  • Tap “Touch to Begin” to start (goes full-screen).
  • Tap the left half of the book to go back a page, the right half to go forward.
  • Use the on-screen arrows or page slider (if enabled) to jump around.
  • (3D only) Tap EN / RO to switch language.

10. Troubleshooting

ProblemFix
An old image still shows after I replaced it Hard refresh the 3D book: Ctrl+Shift+R. The light book updates with a normal refresh.
A white / blank page appears There's a blank image file with that number — delete it from the page folder.
The book stops earlier than expected Check the file numbering. More than 5 missing numbers in a row ends the book. Keep numbers continuous (a single gap is fine).
It opened in the wrong language Set "book-lang" in assets/user-config.json to "ro" or "en".
Full-screen / images didn't work Make sure you launched with the .bat file, not by opening the HTML directly.
PDF converter can't save to the folder Use Google Chrome or Microsoft Edge.
The kiosk feels slow (3D) In the dev panel → VISUAL tab, lower the quality/effects, then Save All.
